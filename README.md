## Customization

### Branding
- Replace `images/kenya-logo.png` with your logo
- Update contact information in all pages
- Modify color scheme in `css/style.css` CSS variables
- Replace placeholder images with your own

### Content
- Update destination information
- Modify service offerings
- Customize pricing and packages
- Add your own testimonials and team information

### Contact Integration
The contact forms are connected to Supabase. To set up:
1. Create a Supabase project
2. Create a table called `contact_inquiries`
3. Update the API key and URL in `js/main.js`
